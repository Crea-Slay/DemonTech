# Introduction

Ce mod pour chaise de bureau est un clone du fameux MonsterTech

Il nécessitera quelques matériaux et/ou peut-être l’achat de machines pour faire les découpes.

Les pièces à découpé sont en HPL de 9mm pour le support chaise et du multiplex 9mm pour les supports joysticks/throttles.

**Attention** : Ce mod fait l’affaire mais n’est pas aussi qualitatif que ceux de MonsterTech. Il y aura un léger battement de gauche à droite. 

# Pré-requis

Pour faire ce mod, vous aurez besoin :

- 1x Planche HPL/Trespa de 8-9mm (idéalement : 40x60 cm)
- 2x Profilé Aluminium 80x40 (23cm) (Attention : ils doivent être taraudé sur un des côtés)
- 2x Profilé Aluminium 80x40 (7cm)
- 2x Equerre 80 rainure T8-10
- ~25 Ecrou avec guidage T8
- ~16 vis M8x 20
- Les vis et écrous correspondant à votre support throttle/Joystick (par exemple : le T16k/visM6, le VKB/vis M5)

Logiciels/Machines :

- Fusion 360 (gratuit)
- CNC avec une fraise de 6 ou 8 mm (voir FABLAB dans votre région)

## (Si pas de CNC)

Peu de gens connaissent le principe de fablab (cf: laboratoire de fabrication).
Pourtant c’est le moyen le plus simple et le moins onéreux pour “produire” une pièce qui n’est pas disponible sur le marché.
A titre d’information, en apportant ma plaque de HPL, il m’a fallu 5min de découpe à la CNC et ptet bien 10-15min de préparation sur le logiciel Fusion 360 (pour rappel, je travaille dans un fablab)

Donc ça peut vous coûter ptet une 20aine d’euros en plus pour faire les découpes.

# 1. Préparation des fichiers à usiner

Maintenant que vous avez vos plaques, il vous faut trouver le modèle 3D.
J’ai répertorié mes fichiers sur un github pour que ce soit plus simple à retrouver :

[https://github.com/Crea-Slay/DemonTech](https://github.com/Crea-Slay/DemonTech)

## Le support de chaise

Sur le site, vous retrouverez les modèles 3D des différents supports de chaises (appelés Form1 à 7).
Pour la majorité des chaises gamings, c’est la Form2 qui est la plus courante. 
(Pour l’instant, seul la Form2 est disponible, mais j’ai récupérer les PDFs avec les cotations des autres formes sur le site de MonsterTech)

Si vous avez une chaise spécifique, vous pouvez aller sur ce lien pour savoir quel forme vous avez besoin :
[https://www.monster.tech/fr/produit/plaque-de-fixation-de-chaise-par-forme/](https://www.monster.tech/fr/produit/plaque-de-fixation-de-chaise-par-forme/)

(généralement, les fablabs et CNC fonctionne avec Fusion360, récupérez le fichier au format .f3d sur le github. Si indication contraire, prenez le fichier au format .stp ou .step )

## Le support Throttle/Joystick

Toujours sur le GitHub, vous retrouverez également les modèles 3D des supports pour votre hotas/hosas en fonction de votre config.
(Pour l’instant, seul le T16k et son throttle TWSC y sont, je rajouterai les autres au fur et à mesures ou à la demande)

# 2. Montage

Quand vous aurez toutes vos pièces, il ne vous restera plus qu’à tout monter sur votre chaise gaming.

1. Coucher votre siège et dévisser les accoudoirs
2. Placer les supports de chaise en desosus des accoudoirs.
(N.B: vous pouvez les mettre entre la plaque de l’accoudoirs et la chaise pour gagner en solidité mais vos accoudoirs seront ptet plus bas. Pour moi qui suit grand, je préfère les laisser dessous que entre)
3. Revisser les accoudoirs.
4. Préparer les profilés. Fixer le petit profilé sur le côté haut du long profilé (partie non-taraudée) avec l’équerre en utilisant les vis M8 et les écrous avec guidage T8. 
5. Placer les profilés de 27cm, côté taraudé contre les supports et les fixer avec les vis M8. (Attention, ne pas serrer SUPER fort au risque de casser le filletage dans le profilé)
6. Mettre les écrous de guidage dans le profilé de 7cm et venir fixer la planche du support Hotas/hosas. 
7. Fixer votre joystick/throttle sur vos planches
8. It’s done !
9. (optionnel : si vous avez trop de flexibilité, vous pouvez ajouter des colsons (serre-cables) autour de vos profilés et les attachés sur les accoudoirs pour gagner en rigidité)
